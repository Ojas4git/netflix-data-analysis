# Netflix Data Analysis

This project performs exploratory data analysis on Netflix movies and TV shows.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Analysis Performed
- Movies vs TV Shows comparison
- Content growth over years
- Top genres on Netflix
- Top directors
- Content added per year

## Key Insights
- Netflix has significantly more movies than TV shows.
- International Movies and Dramas dominate the platform.
- Content production increased rapidly after 2016.
